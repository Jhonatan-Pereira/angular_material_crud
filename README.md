## Getting Started

### Install dependencies
```sh
npm install -g @angular/cli
```

### Install packages
```sh
npm install
```

### Development
```sh
ng serve
```

Clique em [http://localhost:4200/](http://localhost:4200/) para visualizar no browser.

Projeto [backend](https://github.com/Jhonatan-Pereira/node_json_serve).
